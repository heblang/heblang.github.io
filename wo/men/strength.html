Strength Test

Important

Before starting the exercises, we do a strength test to track the results

1. Pull-ups 1 set of maximum reps. Rest for 15 minutes.
2. Parallel bar dips, 1 set of maximum reps. Rest for 20 minutes.
3. Push-ups, 1 round of maximum reps. On the same day, or preferably the next day, start the workouts.
